# Claude Spinners Fortune

A collection of 10,000 Claude Spinner Verbs, extracted from [fortune](https://en.wikipedia.org/wiki/Fortune_(Unix)).

## How to Use

**For Claude (via local config):**

1. Locate the [Claude Code settings JSON](.claude/settings.json) in this repo.
2. Copy the section you want.
3. Paste it into your Claude Code config file (e.g., `.claude/settings.json`).

See docs for additional details: https://code.claude.com/docs/en/settings

**For VSCode Claude Plugin Users:**

1. Locate the [VSCode settings JSON](.vscode/settings.json) in this repo.
2. Copy the section you want.
3. Paste it in your VSCode settings file.
